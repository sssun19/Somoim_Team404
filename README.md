```

```


## 필수
```ruby
$git config --global core.autocrlf true
```


1. 초기화
```ruby
git init
```

2. 파일 모두 추가
```ruby
git add.
```

3.상태 확인 (선택사항)
```ruby
git status
```

4.Commit
```ruby
git commit -m "Commit Name"
```

5. GitHub Push
```ruby
git push origin 브랜치명
```

6. 브랜치 스위치
```ruby
git checkout 브랜치명
```
